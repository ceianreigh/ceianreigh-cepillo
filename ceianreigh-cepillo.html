PUPunta Ride Booking App

PUPunta is a Python ride-booking desktop app for PUP students and commuters. Featuring a Python GUI and integrated HTML map, users can manage accounts, select origins/destinations, choose 2, 4, or 6-seat vehicles, and view pricing.
